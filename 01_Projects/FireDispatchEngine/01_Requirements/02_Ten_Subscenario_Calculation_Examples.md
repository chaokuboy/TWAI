# 10大子场景详细计算示例

**最后更新**：2026-04-23
**标签**：#子场景示例 #N_total #画像计算 #调派编成
**覆盖范围**：95%+ 日常火警
**页面作用**：产品，开发、支队培训的实战计算参考

**扩展公式**（每次计算均严格遵循）：
$$
N_{\text{total}} = N_{\text{base}} + \alpha_{\text{sub}} + \beta_{\text{struct}} + G_{\text{special}} + M_{\text{support}} + \Delta_{\text{confidence}} + \Delta_{\text{scene}}
$$

---

### 1. 普通住宅火灾（占比 ≈42%）

**典型画像**
- 能量物质：A类可燃物（家具/电器）
- 建筑结构：普通多层/低层住宅
- 建筑高度：<24m
- 被困人数：0-5
- 发展阶段：初起/发展

**N_total 计算**
- N_base = 2
- α_sub = 0
- β_struct = 0
- G_special = 0
- M_support = 0
- Δ_confidence = 0
- Δ_scene = 0
- **N_total = 2~4**（映射一级警）

**推荐编成**：2 台水罐主战车 + 1 台指挥车（可选）

---

### 2. 高层住宅火灾（占比 ≈18%）

**典型画像**
- 能量物质：锂电池/电器
- 建筑结构：高层住宅
- 建筑高度：24-100m（重点85m）
- 被困人数：5-30
- 发展阶段：发展→猛烈

**N_total 计算**
- N_base = 4
- α_sub = +2（锂电池）
- β_struct = +2（85m，每30m+1）
- G_special = 0
- M_support = +1（猛烈）
- Δ_confidence = 0
- Δ_scene = +1
- **N_total = 9**（映射三级警）

**推荐编成**：3 台水罐 + 3 台登高平台 + 1 台细水雾 + 1 台抢险救援 + 指挥车 + 保障车
