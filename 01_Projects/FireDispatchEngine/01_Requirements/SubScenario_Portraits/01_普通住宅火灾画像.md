# 01_普通住宅火灾画像模板

**子场景**：普通住宅火灾
**覆盖占比**：42%
**N_total 典型值**：2-4
**映射等级**：一级警

## 典型画像槽位

| 槽位名称     | 典型值                  | 置信度要求 | 备注                     |
|--------------|-------------------------|------------|--------------------------|
| 能量物质     | A类可燃物（家具、电器） | ≥85%       | 极少危化品               |
| 建筑结构     | 普通多层/低层住宅       | 100%       | -                        |
| 建筑高度     | <24m                    | 100%       | -                        |
| 被困人数     | 0-5                     | ≥80%       | -                        |
| 发展阶段     | 初起/发展               | ≥70%       | -                        |

## N_total 计算示例

- N_base = 2
- α_sub = 0
- β_struct = 0
- G_special = 0
- M_support = 0
- **N_total = 2-4**

## 推荐编成

- 2 台水罐消防车（主力）
- 1 台指挥车（可选）

## 触发条件与注意事项

- 报警描述含"家里着火""厨房着火""沙发着火"等
- 低置信度时优先确认被困人数
