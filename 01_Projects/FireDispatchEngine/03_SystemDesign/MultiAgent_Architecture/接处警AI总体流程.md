# 接处警AI总体流程

## 全流程概览

```
接警 → AI辅助录入 → 智能分类 → 调派生成 → 人工确认 → 力量出动 → 动态跟踪
```

## 各环节说明

### 接警

- 接收报警信息（电话、短信等）
- 基础信息记录（时间、地点、联系方式）

### AI辅助录入

- 语音/文本语义理解
- 关键要素自动提取
- 与[[消防多智能体应用]]中的Agent协作

### 智能分类

- 基于[[问题路由树]]的多级分类
- 调用[[多维问题与要素识别]]进行维度建模

### 调派生成

- [[调派业务逻辑与规模核心模型]]生成初版方案
- [[调派引擎约束校验机制]]校验

### 人工确认

- [[人工责任确认模拟]]定义需人工确认的节点
- 值班员确认或修改方案

### 力量出动与跟踪

- 指令下发至消防站
- 实时跟踪力量状态 ^[inferred]
