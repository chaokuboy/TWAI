# 消防多智能体应用

## 应用场景

消防多智能体系统覆盖以下核心场景：

### 接警录入

- AI辅助识别报警人描述的核心事件
- 自动提取时间、地点、类型等关键要素
- 与[[对接处警系统审计]]中描述的119系统对接

### 问题分类

- 基于[[问题路由树]]的多级分类
- 识别火灾等级（[[一级火灾场景全链路模拟|一级]]/[[二级火灾场景全链路模拟|二级]]）
- 关联特殊场景（[[城中村火灾场景全链路模拟|城中村]]/[[高层医院火灾场景全链路模拟|高层医院]]）

### 力量调派

- 根据[[调派业务逻辑与规模核心模型]]生成调派方案
- [[调派引擎约束校验机制]]进行合规性校验
- [[人工责任确认模拟]]处理需人工介入节点

## Agent协作流程

```
意图理解 → 问题分类 → 语义联动 → 调派规划 → 责任确认
```
