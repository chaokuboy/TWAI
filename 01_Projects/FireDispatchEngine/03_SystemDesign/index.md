# 调派引擎模块导航

*最后更新：2026-04-23*

## 模块定位

调派引擎是接处警7.0系统的**核心决策模块**，位于智能分类之后、力量出动之前。

## 核心思维导图

```mermaid
mindmap
  root((调派引擎))
    核心目标
      数据定力
      按出动力度定级
      熟悉度+ETA平衡
    业务逻辑
      队站确定(主管锚定+ETA补齐+梯次)
      车辆确定(功能对口+约束校验)
      规模计算(N_total公式)
    定级与编成
      N_total → 自动定级
      定级反向验证编成
      编成合理性闭环
    保障机制
      约束校验(人员→道路→空防)
      人工确认节点
    闭环优化
      全链路审计
      责任追溯
      报告模板
```
