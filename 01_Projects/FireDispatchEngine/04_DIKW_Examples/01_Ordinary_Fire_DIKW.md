## 场景概述

**普通住宅火灾** —— 最常见的火灾场景，覆盖约 42% 的日常火警。

---

## D → I → K → W 层级转化

### D - Data（数据）

原始报警描述：
> "喂119吗，我家房子着火了，有人在里面，具体不清楚，几楼也不知道"

| 字段 | 值 |
|------|-----|
| 时间 | 14:32 |
| 来源 | 居民报警 |
| 原始描述 | 房子着火，有人被困，具体楼层不明 |

---

### I - Information（信息）

处理后结构化信息：

| 槽位 | 值 | 置信度 |
|------|-----|--------|
| 灾害类型 | 火灾（普通建筑） | 95% |
| 建筑类型 | 普通住宅（砖混/预制板） | 85% |
| 楼层 | 不明 | - |
| 被困人数 | 不明 | - |
| 位置 | 城中村/老旧小区 | 80% |
| 时段 | 白天 | 90% |

---

### K - Knowledge（知识）

触发规则：
- 普通住宅 + 楼层不明 + 被困人数不明 → **谨慎原则**
- 推荐编成：**N_total = 4-6**
- 首波力量：2台水罐车 + 1台抢险车
- 增援预期：通常不需要大量增援

**N_total 计算**：
```
N_base = 4（普通住宅默认）
α_sub = +1（被困人员可能）
β_struct = 0（普通建筑）
G_special = 0（无特殊危险品）
M_support = 0（普通时段）
Δ_confidence = 0
Δ_scene = +1（城中村环境复杂）
----------------
N_total = 6（二级警）
```

---

### W - Wisdom（智慧）

**决策判断**（超越规则）：

| 判断点 | 决策 |
|--------|------|
| 楼层不明 | 架设15米拉梯探查，不盲目内攻 |
| 被困不明 | 坚持"生命优先"原则，优先搜救 |
| 城中村 | 注意道路狭窄，提前规划水源 |

**升级判断**：如报警描述有"学生在补习"、"老人院"等关键词，立即升三级并通知医疗增援。
