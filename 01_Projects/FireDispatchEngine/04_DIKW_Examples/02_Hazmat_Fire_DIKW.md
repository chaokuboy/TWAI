## 场景概述

**危化品火灾** —— 高风险场景，包括化工园区、锂电池、危化品运输等，需要特殊装备和防毒防爆措施。

---

## D → I → K → W 层级转化

### D - Data（数据）

原始报警描述：
> "喂，工业园区里面一个化工厂着火了，有爆炸声，黑烟很大，有人受伤"

| 字段 | 值 |
|------|-----|
| 时间 | 03:17 |
| 来源 | 园区保安报警 |
| 原始描述 | 化工厂着火，有爆炸，黑烟，可能有人员伤亡 |

---

### I - Information（信息）

处理后结构化信息：

| 槽位 | 值 | 置信度 |
|------|-----|--------|
| 灾害类型 | 危化品火灾 | 95% |
| 危险等级 | 极高（爆炸性） | 90% |
| 园区类型 | 化工园区 | 85% |
| 物质类型 | 不明（可能是甲类危化品） | 70% |
| 被困人数 | ≥3（可能有伤亡） | 80% |
| 扩散趋势 | 快速扩散 | 85% |
| 时段 | 深夜 | 95% |

---

### K - Knowledge（知识）

触发规则：
- 化工园区 + 爆炸 + 不明物质 → **最高警戒**
- 推荐编成：**N_total = 13-20**（一级警）
- 首波力量：防毒防爆装备 + 大量泡沫 + 医疗
- 必须联动公安、化工专家、环保部门

**N_total 计算**：
```
N_base = 6（化工园区默认）
α_sub = +3（危化品 + 爆炸）
β_struct = +2（化工装置复杂）
G_special = +5（防毒防爆 + 特殊装备）
M_support = +2（持续供液 + 冷却）
Δ_confidence = 0
Δ_scene = +2（夜间 + 快速扩散）
----------------
N_total = 20（一级警，可能升特级）
```

---

### W - Wisdom（智慧）

**决策判断**（超越规则）：

| 判断点 | 决策 |
|--------|------|
| 物质不明 | 不盲目内攻，等待化工专家确认 |
| 有爆炸 | 划定500米警戒圈，禁止无关人员进入 |
| 深夜时段 | 通知气象部门，评估下风向影响 |
| 可能有伤亡 | 同步通知120，医疗增援待命 |

**降级判断**：如化工专家确认仅为普通塑料燃烧且无毒害，可在评估后逐步降级。

**特别注意**：
- 全程开启可燃气体检测
- 风向变化时立即撤离
- 禁止使用直流射流
