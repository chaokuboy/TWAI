# 多维问题与要素识别

## 三维度模型

### 问题维

描述事件本身的物理属性：
- 火灾类型（建筑、危险物质、森林等）
- 火灾等级（一级、二级）
- 燃烧物质（普通可燃物、危险化学品）

### 意图维

描述报警人的实际诉求：
- 报警目的（请求救援、咨询、举报）
- 紧迫程度（紧急、非紧急）
- 期望结果（人员疏散、火灾扑灭）

### 画像维

描述事件的环境上下文：
- 地理位置（城中村、高层建筑、医院）
- 建筑特征（结构类型、消防设施配置）
- 人员情况（被困人数、易感人群）

## 语义联动机制

三维度信息通过语义联动实现交叉推理，例如：

- 高层医院 + 意图维为紧急 → 推荐优先调派高喷车和救护力量 ^[inferred]
- 城中村 + 问题维为危险物质 → 触发特殊处置流程 ^[inferred]
