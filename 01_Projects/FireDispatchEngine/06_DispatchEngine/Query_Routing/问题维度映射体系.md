# 问题维度映射体系

## 三维映射关系

问题维、意图维、画像维之间的映射构成语义联动的基础：

```
问题维 + 意图维 → 调派优先级
问题维 + 画像维 → 力量编成
意图维 + 画像维 → 特殊需求识别
```

## 典型映射规则

| 问题维 | 意图维 | 画像维 | 映射结果 |
|---|---|---|---|
| 一级火灾 | 紧急 | 高层医院 | 优先调派高喷车+救护 |
| 危险物质 | 举报 | 城中村 | 触发危化处置流程 |
| 二级火灾 | 咨询 | 普通建筑 | 标准处置+信息反馈 |

## 在[[多维问题与要素识别]]中的作用

该体系是语义联动的数学表达，将离散的维度特征转化为可计算的映射关系，支撑[[调派业务逻辑与规模核心模型]]的决策逻辑。
