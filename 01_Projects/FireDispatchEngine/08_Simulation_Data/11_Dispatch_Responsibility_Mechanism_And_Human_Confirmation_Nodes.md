# 调派：责任机制与人工确认节点

## 核心原则

AI系统定位为**辅助决策**，人工保持最终决策权。责任链条：

```
报警人 → 值班员 → 系统 → 值班员 → 力量
```

## 确认节点详细定义

### L0 - 系统自动执行

- 接收报警信息
- 基本要素提取
- 标准场景的常规力量调派（无特殊条件时）

### L1 - 值班员确认

- 火灾等级分类结果
- 力量编成方案
- 出动指令下发

### L2 - 值班长审批

- 跨区增援
- 多起警情的力量分配
- 超标准调派

### L3 - 指挥中心决策

- 区域性大规模调度
- 跨区域多力量协调
