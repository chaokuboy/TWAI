# 02_Areas / 长期维护领域

**标签**：#README
**文件夹职责**：需要持续投入精力维护的**能力领域**，不绑定具体项目

## 子领域说明

- `KnowledgeManagement/`：第二大脑方法论、PARA、DIKW、状态机
- `AI_Model_Dispatch/`：小认知核心、模型路由、Prompt 工程
- `Emergency_Response/`：消防、应急、救援领域专业知识
- `Personal_Productivity/`：个人效率系统、工具链

## Areas vs Projects 区别

- **Projects**：有明确截止日期，当前进行中
- **Areas**：无截止日期，长期维护责任范围

## 维护原则

每月至少复盘一次，重要更新同步到对应 MOC。
