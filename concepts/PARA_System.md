# PARA_System

**PARA** (Projects / Areas / Resources / Archives) 是由 Tiago Forte 在《Building a Second Brain》中提出的**行动导向知识组织方法论**。

## 核心理念

> **知识的价值不在于存储，而在于调用**

## 四大分类

| 分类 | 全称 | 特征 | 示例 | 活跃度 |
|------|------|------|------|--------|
| **P** | Projects | 有明确目标，有时限 | FireDispatchEngine开发 | ★★★★★ |
| **A** | Areas | 长期维护责任范围 | 消防知识、AI调度 | ★★★★☆ |
| **R** | Resources | 参考资料库 | 消防基础知识、模板 | ★★★☆☆ |
| **A** | Archives | 已完成/废弃内容 | 旧项目文档 | ★☆☆☆☆ |

## PARA 流动性

```
Projects → Areas → Resources → Archives
（随项目进程动态流动）
```
