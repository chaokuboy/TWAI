# 事件画像

**事件画像**是警情的**结构化特征描述**，通过槽位填充生成完整的事件画像，是 [[FireDispatchEngine]] 中 [[N_total]] 计算的**唯一输入**。

## 画像生成流程

```
原始报警 → 小认知核心槽位提取 → 事件画像 → 子场景匹配 → N_total计算
```

## 核心槽位定义

| 槽位类别 | 包含字段 | 置信度要求 |
|----------|----------|------------|
| 时间 | 发生时间、发现时间、报警时间 | 高 |
| 地点 | 地址、坐标、所属辖区 | 高 |
| 类型 | 火灾、救援、危化等 | 高 |
| 规模 | 燃烧物质、建筑结构、被困人员 | 中~高 |
| 特殊 | 危化品种类、医疗需求、特种设备 | 视情况 |

## 置信度修正

当核心槽位平均置信度 < 75% 时，触发 [[N_total]] 公式中的 **Δ_confidence** 修正项，自动上浮1~2车。
