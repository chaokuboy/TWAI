# 外部记忆

**外部记忆**是 [[SecondBrain_Agent]] 和 [[FireDispatchEngine]] 的关键基础设施，结合向量检索和知识图谱索引。

## 组成

| 组件 | 功能 | 用途 |
|------|------|------|
| 向量检索 | 相似度匹配 | 检索相似历史案例 |
| 知识图谱索引 | 实体关系查询 | 规则、公式、预案检索 |

## 工作流程

```
小认知核心 → 查询外部记忆 → 返回相似案例/规则 → 辅助决策
```

## 核心作用

- **历史案例匹配**：相似警情 → 参考处置方案
- **规则索引**：快速检索业务规则、计算公式
- **预案调用**：触发对应预案流程
