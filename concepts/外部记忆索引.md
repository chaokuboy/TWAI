# 外部记忆索引

**外部记忆索引**是 [[SecondBrain_Agent]] 外部记忆的分类结构，供小认知核心、RAG系统调用。

## 分类结构

| 分类 | 说明 | 用途 |
|------|------|------|
| FireDispatch_Tags | 消防调派专用向量标签 | 警情匹配 |
| DIKW_Tags | DIKW层级标签体系 | 知识分类 |
| KnowledgeGraph | 实体-关系图谱 | 关联推理 |
| VectorTags | 通用向量检索标签 | 语义搜索 |

## 目标

> 实现任意 Wisdom 内容被小认知核心**秒级精准检索**

## 工作原理

```
小认知核心查询 → 向量匹配 + 图谱推理 → 返回相关Knowledge/Wisdom
```
