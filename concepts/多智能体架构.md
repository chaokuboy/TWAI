# 多智能体架构

**多智能体架构**是 [[FireDispatchEngine]] 的系统架构设计，多个智能体协同工作。

## 智能体类型

| 智能体 | 职责 |
|--------|------|
| 接警智能体 | 接收报警信息 |
| 画像智能体 | 生成事件画像 |
| 计算智能体 | N_total计算 |
| 定级智能体 | 警情定级 |
| 校验智能体 | 约束校验 |
| 确认智能体 | 人工确认协调 |

## 协同流程

```
接警智能体 → 画像智能体 → 计算智能体 → 定级智能体 → 校验智能体 → 确认智能体 → 出动
```
