# 审计日志

**审计日志**是 [[FireDispatchEngine]] 的全链路不可篡改记录，确保责任清晰可追溯。

## 记录内容

| 阶段 | 记录内容 |
|------|----------|
| 接警 | 时间、来源、报警内容 |
| 定级 | N_total计算值、映射依据 |
| 校验 | 约束校验结果 |
| 人工确认 | 确认人、时间、意见 |
| 出动 | 车辆编成、实际到场时间 |
| 战评 | 处置评价、改进建议 |

## 特性

- **不可篡改**：哈希链式存储
- **一键审计报告**：自动生成
- **责任到人**：每个节点记录责任人
