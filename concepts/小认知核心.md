# 小认知核心

**小认知核心**是 FireDispatchEngine 和 [[SecondBrain_Agent]] 的核心组件，负责接收报警信息并提取关键槽位。

## 定位

传统 AI：庞大模型，通用能力强但响应慢、成本高 → **小认知核心**：专注槽位填充，轻量、快速、成本低

## 核心职责

1. **接收原始报警信息**（文字、语音）
2. **提取关键槽位**（时间、地点、类型、规模等）
3. **生成事件画像** → 输入 [[N_total]] 计算
4. **触发 [[DIKW]] 转化流程**

## 与外部记忆的关系

小认知核心调用 [[外部记忆]] 进行：
- 向量检索（相似历史案例）
- 知识图谱索引（规则、公式、预案）

## 核心原则

> 干净数据优先 + 小核心决策 + 外部记忆检索

## 应用场景

- [[FireDispatchEngine]]：119报警处理
- [[SecondBrain_Agent]]：通用知识处理
