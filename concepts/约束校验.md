# 约束校验

**约束校验**是 [[FireDispatchEngine]] 确保**实战可行性**的三层校验机制。

## 三层约束

| 层级 | 约束类型 | 校验内容 | 目标 |
|------|----------|----------|------|
| 第一层 | **人员约束** | 可用消防员数量、资质 | 确保有人 |
| 第二层 | **道路约束** | 路况、距离、拥堵 | 确保能到 |
| 第三层 | **空防约束** | 机场净空、限高 | 确保安全 |

## 校验流程

```
调派方案生成 → 人员约束校验 → 道路约束校验 → 空防约束校验 → 最终方案输出
```

## 目标指标

- **实际到场率 ≥ 95%**
- 资源利用率 85-95%
- 冗余率 < 15%
