# 警情定级

**警情定级**是由 [[N_total]] 反向映射生成的1-5级分类体系，是 [[FireDispatchEngine]] "**按出动力度定级**"核心原则的实现。

## 5级分类

| 等级 | N_total范围 | 典型场景 | 联动级别 |
|------|-------------|----------|----------|
| **一级警** | 1~4车 | 普通住宅火灾 | 辖区内快速响应 |
| **二级警** | 5~8车 | 地下车库初期、电动车火灾 | 支队增援 |
| **三级警** | 9~12车 | 高层住宅、锂电池仓库 | 跨区调派+特种 |
| **四级警** | 13~18车 | 化工园区、工业厂房 | 全市联动+专家组 |
| **五级警** | 19车+ | 大面积危化泄漏+多人被困 | 全省/跨省联动 |

## 定级流程

```
事件画像 → N_total计算 → 查映射表 → 生成警情等级 → 定级反向验证 → 输出最终方案
```

## 定级反向验证

定级生成后进行**反向验证**，确保编成与等级相互印证：
- 合理 → 输出最终方案
- 不合理 → 自动上浮一级 + 增援 + 人工确认

## 特殊情况

- **置信度 < 80%**：必须人工确认后再定级
- **多警情并发**：可临时下调非关键警情等级（需记录理由）
- **现场反馈**：火势蔓延时自动上浮一级
