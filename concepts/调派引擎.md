# 调派引擎

**调派引擎**是 [[FireDispatchEngine]] 接处警7.0系统的**核心指挥大脑**，负责从事件画像完成到最终出动方案生成的完整闭环。

## 核心流程

```
接警 → 画像生成 → N_total计算 → 定级映射 → 反向验证 → 出动方案 → 战评
```

## 架构组件

| 组件 | 功能 |
|------|------|
| 子场景匹配 | 匹配10大子场景 |
| N_total计算引擎 | 核心公式计算 |
| 规则引擎 | 业务规则执行 |
| 约束校验引擎 | 三层校验 |
| 定级反向验证 | 确保等级合理 |
| 人工确认中心 | 关键节点确认 |
| 出动方案生成 | 最终编成输出 |
| 审计日志服务 | 全链路记录 |

## 核心原则

> 干净数据优先 + 小核心决策 + 外部记忆检索

## 性能指标

- 全流程 ≤ 300ms
- 支持 100+ 路并发
- N_total准确率 ≥ 95%
- 实际到场率 ≥ 95%
