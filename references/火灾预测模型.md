# 火灾预测模型

**火灾预测模型**是用于预测火灾发展趋势的数学模型，是 [[FireDispatchEngine]] 的辅助决策工具。

## 预测要素

| 要素 | 说明 |
|------|------|
| 燃烧物质 | A类(固体)、B类(液体)、C类(气体) |
| 建筑结构 | 钢结构、混凝土、木结构 |
| 发展阶段 | 初起、猛烈、下降 |
| 蔓延风险 | 水平和垂直蔓延 |

## 预测维度

- **火势发展预测**：基于燃烧物质和供氧条件
- **人员风险预测**：被困人数、疏散难度
- **蔓延预测**：基于建筑结构和风速
